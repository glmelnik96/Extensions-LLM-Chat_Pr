# Контекст для следующего агента

## Как пользоваться этим файлом

- **По просьбе пользователя в конце сессии** (или при передаче задачи) агент **дополняет или правит** раздел «Журнал» ниже: дата, что сделано, открытые риски, что проверить в Premiere.
- Не раздувать: только факты, пути к файлам, лимиты (например `maxSteps`), известные баги.
- Остальная «вечная» архитектура — в [PROJECT.md](PROJECT.md).

---

## Снимок продукта (редко менять)

- **Репозиторий:** CEP 12, Premiere Pro 2025+, папка `Extensions-LLM-Chat_Pr` в каталоге CEP extensions.
- **ИИ:** Cloud.ru Evolution Foundation Models — `chat/completions` + `audio/transcriptions` (`client/shared/cloudru-client.js`). Ключ: `client/shared/fm-secrets.js`. Остальное: `fm-defaults.js`.
- **Три панели:** таймкоды | текст (транскрипт + вырезание) | маркеры. См. таблицу в [PROJECT.md](PROJECT.md).

## Быстрый чек перед правками

1. `npm test` — валидаторы инструментов.  
2. После изменений хоста — перезагрузка панели в Premiere, тест на копии секвенции.  
3. Сеть FM: 401/413 — текст в панели и `cloudru-client.js` (413 → меньше чанк / In–Out).
