# Ручные тесты плагина в Premiere Pro

Документ для валидации текущего состояния.
Одна панель «ИИ: монтаж» с двумя вкладками: **Чат** и **Инструменты**.

## Подготовка

1. Перезагрузить плагин: Window → Extensions → ИИ: монтаж
2. Секвенция с 3–5 клипами на V1/A1, длительностью 1–3 минуты
3. `fm-secrets.js` содержит валидный API-ключ Cloud.ru
4. Все модели = `openai/gpt-oss-120b` (chatModel, analysisModel, fastModel)
5. **Транскрибировать** секвенцию (In–Out) для получения `audioAnalysis.silences`

### 2.2. ✂ Убрать паразиты

Два режима: **Строгий** (э, мм, ну, блин) vs **Расширенный** (+ типа, вот, значит, как бы, короче).

| # | Шаг | Ожидание | OK |
|---|------|----------|-----|
| 1 | Toggle «Строгий» (default) | Подсвечен синим | [ ] |
| 2 | Нажать «Найти и вырезать» | Только чистые изолированные филлеры (≤1.5с, 1–4 слова) | [ ] |
| 3 | Проверить: строгий не ловит «вот», «типа» | Расширенные слова пропущены | [ ] |
| 4 | Toggle «Расширенный» | Переключается | [ ] |
| 5 | Нажать «Найти и вырезать» | Больше: + «типа», «вот», «значит», «как бы» (≤2.5с) | [ ] |
| 6 | Строгий ≠ Расширенный | Расширенный находит БОЛЬШЕ паразитов | [ ] |
| 7 | Применить | Паразиты вырезаны, речь не обрывается | [ ] |
| 8 | Нет паразитов в транскрипте | Сообщение: «Whisper может объединять паразиты…» | [ ] |
| 9 | Длинные сегменты с текстом-паразитом | НЕ вырезаны (>1.5с strict, >2.5с normal) | [ ] |

### 2.5. 🎬 J-cuts / L-cuts — ОТКЛЮЧЕНО

| # | Шаг | Ожидание | OK |
|---|------|----------|-----|
| 1 | Карточка J-cuts | Затемнена (opacity:0.5), pointer-events:none | [ ] |
| 2 | Текст ошибки | «Временно недоступно. Связанные клипы обрезаются вместе…» | [ ] |
| 3 | Через чат `/j_cuts` | Ошибка «J/L-cuts временно отключены…» | [ ] |

> J/L-cuts требуют клипы на V1/A1 + V2/A2 с перекрытием.
> ExtendScript не поддерживает `unlink()`.

## 5. Edge cases
